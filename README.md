# Mister Lanches - Full Stack & Framework Comparison Study

Este projeto é o ápice do Portfólio do 8º Semestre Centro Universitário - Católica de Santa Catarina em Jaraguá do Sul. Ele consiste em um sistema completo de delivery para o estabelecimento **Mister Lanches** e um estudo comparativo de performance e engenharia de software.

## 🎯 Objetivo
Desenvolver uma solução robusta para gestão de pedidos e vendas, enquanto fornece dados técnicos comparativos sobre frameworks web baseados em critérios de:
- Desempenho e Testes de Estresse
- Velocidade de Desenvolvimento
- Segurança e Portabilidade

## 📊 Documentação de Engenharia
Os diagramas de arquitetura e casos de uso guiam o desenvolvimento deste MVP:
- **Casos de Uso:** Login, Definição de Pedido, Finalização e Gestão Administrativa.
- **Fluxo de Atividades:** Processo completo desde o cadastro do usuário até a notificação do administrador.

## 🛠️ Requisitos do Sistema
### Funcionais (Principais)
- [x] Cadastro e Login de Usuários/Administradores.
- [x] Gestão de Cardápio (CRUD de produtos).
- [ ] Fluxo de Pedidos (Carrinho, Status e Notificações).
- [ ] Gestão de Estabelecimento.

### Não Funcionais (Critérios de Avaliação)
Conforme as métricas do projeto, a aplicação é avaliada por: **Desempenho (CPU/RAM), Segurança, Documentação e Suporte.**

## 💻 Especificações Técnicas (Ambiente de Testes)
Para garantir a isonomia dos testes comparativos, foi utilizada a seguinte infraestrutura:
- **Hardware:** Intel Core i7-11800H | 16GB RAM | SSD 1TB
- **OS:** Windows 10
- **Stack Principal:** Vue.js, Node.js, MySQL (via Laragon).
